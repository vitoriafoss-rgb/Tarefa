# LocalLink 🧺 - Fortalecendo a Economia do Bairro

O **LocalLink** é uma ponte tecnológica entre o pequeno produtor local e o consumidor consciente. Nossa plataforma elimina as taxas abusivas de marketplaces tradicionais e simplifica a logística de proximidade.

## 🚀 Funcionalidades Principais
- **Vitrine Digital:** Catálogo PWA (Progressive Web App) leve e rápido.
- **Pedido Integrado:** Checkout direto para o WhatsApp do vendedor.
- **Rotas Inteligentes:** Otimização de entregas por geolocalização.
- **Modelo Taxa Zero:** O produtor retém 100% do lucro, pagando apenas uma assinatura fixa.

## 🛠️ Stack Tecnológica
- **Frontend:** React.js (PWA)
- **Backend:** Node.js
- **Banco de Dados:** PostgreSQL
- **Infra:** AWS / Google Cloud

## 📈 Impacto Gerado
- **+30%** de lucro real para o produtor.
- **-20%** de desperdício de alimentos via assinaturas recorrentes.

## 💻 Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/seu-usuario/locallink.git`
2. Instale as dependências: `npm install`
3. Inicie o servidor de desenvolvimento: `npm start`
